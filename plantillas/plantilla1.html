Primer Template Creado con django

Podemos obtener datos desde variables:

{{nombre_persona}} {{apellido_persona}}


Fecha y hora actual: {{la_fecha}}

Solo el día: {{la_fecha.day}}

Día, mes y año: {{la_fecha.day}}/{{la_fecha.month}}/{{la_fecha.year}}


Podemos obtener datos desde listas:

Toda la lista: {{temas}}

Un elemento de la lista: {{temas.1}}


Podemos obtener datos desde clases:

Usuario: {{clave_usuario1}}. Cargo: {{clave_cargo1}}. Salario: {{clave_salario1}}

Usuario: {{clave_usuario2}}. Cargo: {{clave_cargo2}}. Salario: {{clave_salario2}}


Podemos obtener datos utilizando ciclos:

Formato horizontal:

{% for elem in temas %} {{elem}} {% endfor %}

Formato vertical:

{% for elem in temas %}

{{elem}}

{% endfor %}

Formato lista:

    {% for elem in temas %}
  • {{elem}}
    • {% endfor %}

Ejemplo utilizando ciclo if:

    {% if temas %} {% for elem in temas %}
  • {{elem}}
    • {% endfor %} {% else %}

    No hay elementos por ahora.

    {% endif %}
